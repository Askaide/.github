## Purpose

This repository exists to make the official Askaide GitHub organization understandable, crawlable and safe:

- official brand description;
- public security policy;
- public support information;
- AI/entity context;
- mobile app public notes;
- privacy and publication boundaries.

## Publication Rules

Do not publish:

- private application source code;
- production secrets;
- `.env` files;
- Android signing files or keystores;
- API keys;
- user data;
- internal operational documents;
- sensitive business logic;
- private infrastructure details.

Only public, non-sensitive documentation belongs here.
