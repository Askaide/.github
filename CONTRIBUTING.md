# Contributing

Thank you for your interest in Askaide.

This public GitHub organization is primarily used for public documentation, security information, mobile app notes and selected open-source utilities when appropriate.

## What Belongs Here

- public documentation improvements;
- typo fixes in public docs;
- security policy improvements;
- public utility suggestions;
- accessibility or documentation feedback.

## What Does Not Belong Here

- private application source code;
- production credentials;
- `.env` files;
- Android signing files;
- user data;
- account-specific support;
- sensitive operational details;
- unreleased business logic.

## Pull Requests

If public repositories are opened for contribution in the future, pull requests should be small, clear and limited to the documented public scope.

By contributing, you confirm that your contribution does not contain confidential information, private user data or third-party content you do not have permission to share.
