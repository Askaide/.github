# Code Of Conduct

Askaide expects respectful, useful and good-faith participation in public project spaces.

## Expected Behavior

- Be respectful and constructive.
- Stay relevant to the public documentation or project scope.
- Report security issues responsibly.
- Avoid sharing personal data or private account details in public spaces.

## Unacceptable Behavior

- Harassment or abusive language.
- Spam or promotional flooding.
- Publishing private user data.
- Sharing secrets, credentials or internal operational material.
- Encouraging misuse of the platform.

## Enforcement

Askaide may remove inappropriate public content, close issues or restrict participation when needed to keep public spaces safe and useful.
