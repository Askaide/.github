# Public GitHub Publication Policy

This organization profile is intentionally public and intentionally limited.

## Allowed

- public brand information;
- public security policy;
- public support information;
- public AI/entity context;
- high-level mobile app notes;
- selected open-source utilities if approved later.

## Not Allowed

- backend source code;
- marketplace ranking logic;
- production infrastructure details;
- private operational reports;
- secrets or credentials;
- signing files;
- user data;
- private support data;
- unpublished business logic.

## Review Rule

Before publishing new public material, confirm that it is:

- non-sensitive;
- useful to public users or crawlers;
- consistent with Askaide branding;
- not exposing private implementation details;
- not creating false claims.
