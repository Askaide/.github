# Privacy And Data Boundaries

Askaide public GitHub repositories must not contain personal data or operational secrets.

## Do Not Publish

- user names, emails, phone numbers or addresses;
- account exports;
- chat logs or support tickets;
- payment details;
- access tokens;
- OAuth credentials;
- API keys;
- private infrastructure details;
- Android signing files;
- `.env` files;
- database dumps.

## Public Documentation May Include

- public company descriptions;
- support links;
- high-level service descriptions;
- general security guidance;
- public mobile app notes;
- AI/entity context;
- public brand profile.

## Data Requests

Privacy or personal data requests should go through the official Askaide website and support channels, not GitHub.
