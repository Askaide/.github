# sameAs Readiness

This document tracks whether the public GitHub organization is ready to be used as a `sameAs` signal in Askaide structured data.

## Candidate

```text
https://github.com/Askaide
```

## Required Checks

- Organization exists publicly: yes
- Public profile repository exists: yes
- Public profile README exists: yes
- Brand description is coherent: yes
- Website is set to `https://askaide.com`: yes
- Location is set to `Belgium`: yes
- Public logo asset exists: yes
- Private source code is not exposed: yes
- Public docs contain no secrets: yes

## Remaining Human Check

Confirm in the GitHub web interface that:

- the organization avatar displays the official Askaide logo;
- the public overview page displays the organization profile README;
- no unintended screenshot or private material is visible.

## Deployment Rule

Only add `https://github.com/Askaide` to Organization schema after the human check is complete.
