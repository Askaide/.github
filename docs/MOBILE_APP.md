# Askaide Mobile App

Askaide may provide a mobile application experience that connects to the public Askaide web platform.

## Public App Principle

The mobile application should reflect the live public Askaide service while respecting platform rules, user privacy, security and marketplace safety.

## Sensitive Files Not Published

Do not publish:

- Android upload keystores;
- signing passwords;
- Play Console credentials;
- private API keys;
- production environment files;
- user data;
- internal release credentials.

## Public Information Allowed

Public documentation may describe:

- app purpose;
- supported countries;
- high-level marketplace functionality;
- support links;
- privacy and security principles;
- official website links.
