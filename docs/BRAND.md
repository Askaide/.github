# Askaide Brand Profile

## Short Description

Askaide is a Belgian-first local services marketplace connecting individuals with trusted local service providers.

## Long Description

Askaide helps people find nearby support for everyday home and local service needs, including cleaning, handyman work, gardening, childcare, tutoring, moving help, transport, senior support and computer assistance.

The platform is anchored in Belgium and supports selected multi-country coverage in France, the Netherlands, Germany, Spain and Morocco.

## Official Entity Statement
